# SPAwN-docker

This repository offers docker environment for [SPAwN](https://gitlab.com/UnBVision/spawn/-/tree/main)

## Install

1. Clone

```shell
git clone --recursive https://github.com/Decwest/SPAwN-docker.git
```

2. Build docker image

```shell
cd SPAwN-docker
./build.sh
```

3. Run docker container

```shell
cd SPAwN-docker
./run.sh
```
